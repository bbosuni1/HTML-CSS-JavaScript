선택자 { font=family : 글꼴 명 }

선택자 {font-size : 크기;}

글꼵 굵기 설정하기

선택자 { font-weight : normal | blod | bolder | lighter }
  • [굵게] : 언젠가 너와 나 남이 되어도
  • [더굵게] : 어딘선가 누군가에 무슨일이 생기면
  • [가늘게] : 크으대 키어키 취이난 솨랑이

font-variant : 영대문자를 작게 표현

선택자{font-variant : 속성값;}

[기본글꼴] WE WILL FIND A WAY. WE ALWAYS HAVE.

[변경글꼴] we will find a way. we always have.

font-style : 폰트의 기울기 설정

선택자 { font-style : normal | italic | oblique; }
  • [기본] : 자 이제 시작이야 (내꿈을), 내 꿈을 위한 여행 (피카츄)
  • [italic] : 자 이제 시작이야 (내꿈을), 내 꿈을 위한 여행 (피카츄)
  • [oblique] : 자 이제 시작이야 (내꿈을), 내 꿈을 위한 여행 (피카츄)

    • font : 폰트 관련 설정을 종합하여 한 번에 적용할 수 있다.

                     font-variant
