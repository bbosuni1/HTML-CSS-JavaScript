속성 선택자

기본 선택자 뒤에 []를 붙여
특정 속성을 가진 태그들만 선택하는 방법. 

            기본 형식 -> 선택자[속성=속성값] { . . .}
            =   :   속성값과 같은 속성값을 가질 때
            ~=  :   특정 단어를 속성값으로 가질 때
            ^=  :   속성값이 특정 단어로 시작할 때
            $=  :   속성값이 특정 단어로 끝날 때
            *=  :   속성값이 특정 단어를 포함할 때

자손 선택자와 후손 선택자

자손 선택자 : 바로 안에 있는 태그
후손 선택자 : 안에 존재하는 모든 태그들 

            자손 선택자 : 부모선택자>자손선택자 { . . .}
            후손 선택자 : 부모선택자 자손선택자 { . . .}

나는 자손1 입니다.

나는 자손2 입니다.

    나는 자식 있는 자손3 입니다.
  • 나는 ul의 자손임과 동시에 div의 후손1 입니다.
  • 나는 ul의 자손임과 동시에 div의 후손2 입니다.
  • 나는 ul의 자손임과 동시에 div의 후손3 입니다.

동위 선택자

동급 관계에 있는, 즉 한 부모 아래 같이 있는 태그들 끼리 선택하는 방법

            선택자A+선택자B {. . .} : A요소 바로 뒤의 B요소를 선택함
            선택자A~선택자B {. . .} : A요소 뒤의 모든 B요소를 선택함

반응 선택자

사용자가 움직임(마우스,키보드 입력)을 보일 때,
해당 움직임에 따라 스타일이 변하게 하는 것

            선택자:active{ . . .} 사용자가 마우스로 클릭했을 때
            선택자:hover{ . . .} 마우스 커서가 해당 요소 위에 올라 갔을 때
active 테스트 입니다.
hover 테스트 입니다.
