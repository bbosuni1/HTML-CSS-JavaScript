자바 스크립트 개요

스크립트 언어란?
사용자의 요구사항에 맞게 수행 코드를 간결하게 구현할 수 있는 언어.
개발자가 매우 빠르고 쉽게 배울 수 있으며, 짧은 코드로 내용이 구현되기 때문에 생산성도 높다.
자바 스크립트란?
웹 브라우저에서 가장 많이 사용하는 인터프리터 방식의 객체 지향언어
ECMA 스크립트 표준을 따르며 오늘날 웹 구현 스크립트 언어의 대표로 자리잡고 있다.
사용 방법
  1. inline 방식 : 태그 안에 직접 작성하는 방법(간단한 소스코드 사용시)
  2. internal 방식 : script 태그를 별도로 선언하고 그 안에 소스코드를 작성하는 방법
  3. extetnal 방식 : 외부의 .js 파일을 가져와 현재 웹 문서에 등록하는 방식
자바 스크립트 작성 시 주의할 점
자바스크립트에서의 소스코드는 함수 단위로 작성하고, 이벤트 속성을 사용해 실행한다.
이벤트는 각 태그마다 지원하는 차이가 있다.
일반적인 실행 방식은 on이벤트명 = '실행할 함수명([전달인자]);'
