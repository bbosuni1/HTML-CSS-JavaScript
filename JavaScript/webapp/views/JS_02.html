innerHTML을 활용하여 요소의 내용 변경하기

자바 스크립트에서 태그 앨리먼트(요소)의 값을 읽어 오거나, 변경할 수 있다.


console.log() 함수 사용하기

console.log()로 작성한 내용들을 개발자 도구로 확인할 수 있다.
개발자 도구 : 웹 문서의 내용을 소스코드 형태로 볼 수 있는 간이 개발 도구를 말한다
이러한 개발도구를 활용하여 콘솔화면의 특정 값을 수정하거나 특정 로직을 디버깅할 때 사용할 수 있다.
