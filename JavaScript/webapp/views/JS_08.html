자바스크립트 배열

자바 스크립트 배열 :

자바에서는 배열을 선언할 때 특정 자료형만 선언하여 해당하는
값들만 받을 수 있었다. ( ex. int[] --> int값만 저장 가능 )
하지만, 자바 스크립트는 지정하지 않기 때문에,
모든 자료형을 다 받을 수 있다. (자바의 컬렉션과 유사하다)

배열 선언

배열을 선언할 때에는 new Array를 사용할 수도 있고,
[] 를 사용할 수도 있다. 또한 배열 선언시엔 크기를 지정하거나
지정하지 않아도 된다.

Array 객체의 메소드(함수)

Array도 하나의 객체이기 때문에 여러 종유의 함수를 가지고 있다.

indexOf() : 배열의 특정 요소가 몇번째 인지 확인하는 함수

concat() : 두 개의 배열을 하나로 합치는 함수

join() : 배열의 내용을 하나로 합치는 함수

reverse() : 배열의 순서를 거꾸로 뒤집는 함수

sort() : 배열을 오름차순으로 정렬하는 함수

toString() : 배열을 문자열로 바꾸는 함수

push() / pop() :

push() : 배열의 맨 뒤에 값을 추가하는 함수
pop() : 배열의 맨 뒤에 값을 하나 꺼내면서 제거하는 함수

unshift() / shift() :

unshift() : 배열의 맨 앞에 값을 추가하는 함수
shift() : 배열의 맨 앞의 값을 제거하는 함수

slice() / splice() :

slice() : 배열의 요소를 선택하여 부분적으로 잘라내는 함수
splice() : 배열의 인덱스 위치에 요소를 추가하거나 제거하는 함수
