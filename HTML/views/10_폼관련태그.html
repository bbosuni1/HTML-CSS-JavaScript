text 관련 input 태그들

type = "text"

한 줄 짜리 텍스트 데이터를 작성할 때 쓰인다

type = "password"

비밀번호를 입력할 때 쓰인다.

type = "email"

이메일 형식이 올바른 지 검사할 때 사용

type = "search"

검색 전용 박스를 제공하는 속성

type = "url"

입력한 값을 value 값과 합하여 웹 문서를 호출하는 속성 ( 잘 사용하지 않음 )

type = "submit"

 

숫자와 관련된 input태그

type = "number"

입력창에 스핀 형태로 표현되나, 웹 브라우저에 따라 안되는 경우도 있다.

type = "range"

슬라이드 바를 통해 숫자를 지정할 수 있다.

날짜와 관련된 input 태그

type = "date | month | time "
