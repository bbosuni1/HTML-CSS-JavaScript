alt 속성

화면에 이미지를 표기하거나, 이미지가 표시되지 않을 때 해당 이미지를 간략하게 설명하는 속성. 웹 브라우저에서 화면 낭독기(스크린 리더)를 통해 화면의 내용을 읽을 때 사용한다. 화면 낭독기는 시각 장애인, 지체 장애인들을 위한 웹 접근성 프로그램으로 화면의 내용을 보지 못하거나 마우스를 사용하기 힘든 사람들에게 해당 화면을 낭독하여 설명해주는 프로그램이다.

width와 height 속성

사진의 높이와 길이를 설정하는 속성. 사진의 특징을 결정하는데 고정 크기 및 가변 크기설정이 가능.
