속성 선택자 :

선택자[속성명] : 해당 속성을 가진 태그(요소)를 선택
선택자[속성명=속성값] : 해당 속성의 특정 값을 가진 요소 선택
선택자[속성명~=속성값] : 해당 속성의 값 중 특정 값이 있는 요소 선택
선택자[속성명$=속성값] : 속성 안의 값이 '속성값'으로 끝나는 요소
선택자[속성명*=속성값] : 속성의 값 중 특정 값을 포함하는 요소 선택
