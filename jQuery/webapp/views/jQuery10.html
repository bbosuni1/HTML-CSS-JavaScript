jQuery 요소 추가 관련 메소드

추가 메소드 part.1

$('A').appendTo(B) : A를 B요소 안의 마지막에 추가
$('A').prependTo(B) : A를 B요소 안의 앞에 추가
$('A').insertAfter(B) : A를 B요소 뒤에 추가
$('A').insertBefore(B) : A를 B요소 앞에 추가

추가 메소드 part.2

$('A').append(B) : B를 A요소 안의 마지막 부분에 추가
$('A').prepend(B) : B를 A요소 안의 첫 부분에 추가
$('A').after(B) : B를 A요소 뒤에 추가
$('A').before(B) : B를 A요소 앞에 추가

객체(요소) 복사와 제거

clone() : 지정한 객체(요소)를 복사할 때 사용하는 함수
remove() / detach() / empty() : 지정한 객체 요소를 제거할 때 사용하는 함수
