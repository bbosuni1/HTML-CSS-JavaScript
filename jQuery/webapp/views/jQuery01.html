jQuery 개요

jQuery 란?

존 레식에 의해 개발된 JS 경량화 라이브러리로,
복잡한 자바스크립트 구문을 짧고 쉽게 사용할 수 있도록 구현된 라이브러리이다.
-- jQuery 사용 시 장점 --
1. DOM과 관련된 스크립트를 보다 쉽게 구현 가능
2. ajax (비동기 통신), 이벤트 처리를 폭 넓게 지원한다.
3. jQuery 연결할 수 있는 확장형 플러그인, 오픈API를
다양하게 지원한다. (차트, 슬라이드, 애니메이션, 이펙트)

jQuery 연결하기

1. jQuery 홈페이지에서 JS 파일을 다운받아 연결하는 방법
2. CDN(Content Delivery Network)으로 파일 다운로드 없이 연결하는 방법

$('document').ready()

document.onload 와 같이 현재 웹 문서를 로드할 때 실행할 함수를 구현할 수 있다.
